Question 1: How to select an element in the DOM?

You can use methods like:

  • getElementById()
  • getElementsByClassName()
  • querySelector()

Question 2: What is the difference between innerHTML, innerText, and textContent?

  • innerHTML: Includes HTML content.
  • innerText: Includes only visible text.
  • textContent: Gets the text content .

Question 3: What is event delegation?

Event delegation is adding one event listener to a parent element to handle events from child elements.

Question 4: What is event bubbling?

Event bubbling is when an event starts at the target element and then bubbles up to its parent elements.

Question 5: How to create, add, and remove elements?

  • Create: document.createElement()
  • Add: appendChild()
  • Remove: removeChild()
